# 🔀 代码合并辅助工具

> 专为金融合规环境设计的本地代码差异对比工具

## ⚡ 快速开始

```bash
cd code_merge_assistant

# 方式 1：快速演示（无需安装依赖）
python3 demo.py

# 方式 2：完整功能（推荐）
pip3 install -r requirements.txt
python3 web_ui.py
# 浏览器访问 http://localhost:5678
```

## 🎯 解决什么问题？

在微众银行等金融机构的封闭环境中：
- ❌ 代码只能通过企业微信文本传输
- ❌ 无法使用 Git、SVN 等工具
- ❌ 肉眼对比代码，效率低、易出错
- ❌ 手动合并代码，耗时 20-30 分钟

**使用本工具：**
- ✅ 3 秒识别所有代码变更
- ✅ 彩色高亮显示差异
- ✅ 完全本地运行，符合合规要求
- ✅ 效率提升 10 倍（从 30 分钟到 3 分钟）

## 🌟 核心功能

- **智能差异分析**：精确识别新增、删除、修改的代码
- **彩色可视化**：绿色=新增，红色=删除，黄色=修改
- **AST 语法解析**：理解代码结构（类、方法、函数）
- **Web 界面**：美观直观的浏览器界面
- **命令行工具**：适合终端操作
- **跨平台支持**：macOS 和 Windows

## 💡 实际使用场景

### 场景：企业微信代码传输
1. 同事通过企业微信发来代码
2. 打开 Web 界面
3. 左侧粘贴你的代码，右侧粘贴同事的代码
4. 点击"对比分析"
5. 3 秒看到所有差异，快速合并

**效果：从 30 分钟缩短到 3 分钟！**
