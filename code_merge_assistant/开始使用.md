# 🚀 开始使用

## 三种使用方式

### 方式 1：快速演示（无需安装依赖）⚡

最快体验工具功能，无需安装任何依赖：

```bash
cd code_merge_assistant
python3 demo.py
```

这会展示一个简单的代码对比示例，让你快速了解工具的功能。

---

### 方式 2：Web 界面（推荐）⭐

**最直观、最好用的方式！**

#### 第一步：安装依赖
```bash
pip3 install -r requirements.txt
```

如果安装慢，使用国内镜像：
```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 第二步：启动服务
```bash
# macOS/Linux
./start.sh

# Windows
start.bat

# 或手动启动
python3 web_ui.py
```

#### 第三步：使用
1. 浏览器访问：`http://localhost:5678`
2. 左侧粘贴你的代码
3. 右侧粘贴同事的代码
4. 点击"对比分析"
5. 查看彩色高亮的差异结果

**特点：**
- ✅ 彩色高亮（绿色=新增，红色=删除）
- ✅ 实时统计
- ✅ 界面美观
- ✅ 操作简单

---

### 方式 3：命令行工具

适合喜欢命令行的开发者。

#### 对比两个文件
```bash
python3 merge_assistant.py compare file1.java file2.java
```

#### 交互式输入
```bash
python3 merge_assistant.py compare-text
```

然后按提示粘贴代码。

## 实际使用场景

### 场景：同事通过企业微信发来代码

**步骤：**

1. **启动工具**
   ```bash
   python3 web_ui.py
   ```

2. **打开浏览器**
   访问 `http://localhost:5678`

3. **粘贴代码**
   - 左侧：你当前的代码
   - 右侧：同事发来的代码

4. **点击对比**
   点击"对比分析"按钮

5. **查看结果**
   ```
   📊 变更统计：
     • 新增：5 行
     • 删除：1 行
     • 修改：2 行
   
   🔍 详细差异：
   + [行 3] import java.util.Optional;
   - [行 10] return null;
   ⚠ [行 15] 修改方法签名
   ```

6. **快速合并**
   根据高亮的差异，快速完成代码合并

**效果：**
- 传统方式：20-30 分钟
- 使用工具：2-3 分钟
- **效率提升：10 倍！**

## 常见问题

### Q: 端口被占用怎么办？

编辑 `web_ui.py`，修改最后一行：
```python
app.run(host='0.0.0.0', port=8888, debug=True)  # 改成其他端口
```

### Q: 支持哪些编程语言？

支持所有文本格式的代码：
- Java ✅
- Python ✅
- JavaScript / TypeScript ✅
- C / C++ ✅
- Go ✅
- 其他任何文本文件 ✅

### Q: 代码会被上传吗？

**不会！** 所有处理都在本地完成，完全离线运行。

### Q: 可以在公司内网使用吗？

**可以！** 工具完全本地运行，不需要联网。
